
基本命令

### 列出所有的box

```
vagrant box list
```


### vm重启

```
vagrant reload
```

vagrant halt 停止一个虚拟机
vagrant init 初始化一个虚拟机
vagrant package 打包一个虚拟机
vagrant reload 重启一个虚拟机
vagrant ssh ssh进虚拟机
vagrant up 启动一个虚拟机
vagrant status 查看虚拟机状态
